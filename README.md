## 1. 📖 Sobre o Projeto

O **Financial Educator React** é uma aplicação web desenvolvida para
promover educação financeira de forma prática, intuitiva e acessível.

O projeto foi desenvolvido em **React**, com foco em arquitetura
escalável, componentes reutilizáveis, boas práticas de desenvolvimento e
integração com **Firebase**.

## 3. 🚀 Como Acessar

### Clonar o projeto

``` bash
git clone https://github.com/Domisnnet/Financial-Educator-React.git
```

``` bash
cd Financial-Educator-React
```

``` bash
npm install
```

``` bash
npm run dev
```
