# **App Name**: EcoWise AI

## Core Features:

- Live Spend Dashboard: Visual summary of monthly limits, real-time balance tracking, and category-based spending breakdowns.
- One-Tap Expense Logger: Simplified interface for quickly recording daily transactions like coffee or transportation.
- AI Prosperity Planner: An LLM-powered tool that acts as a financial architect to build realistic saving plans based on user income and goals.
- Behavioral AI Nudges: Smart notifications tool that reasons through spending history to send amigable warnings when budget limits are nearing.
- Financial Vision Board: Gamified progress tracking for long-term saving goals, visualizing progress towards purchases like laptops or trips.
- FinEdu AI Tutor: An interactive tool for students to ask complex financial questions and receive answers in simple, jargon-free language.

## Style Guidelines:

- Primary Color: Energetic Indigo (#6B63F5) reflecting a modern and trustworthy financial personality for the younger generation.
- Background Color: Soft Lavender Tint (#F9F8FE) to maintain a clean, high-end agency aesthetic without harsh whites.
- Accent Color: Azure Blue (#3A8DF1) used for interactive elements and AI-driven success indicators.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward vibe. Body font: 'Inter' (sans-serif) for neutral, machined precision in financial data.
- Clean, thin-line glyphs with slightly rounded terminals to convey approachability and simplicity.
- Bento-grid style cards with generous white space, emphasizing visual clarity and responsive adaptation.
- Gentle spring transitions for balance updates and micro-interactions when registering new expenses.
